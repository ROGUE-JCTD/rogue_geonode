State GeoNode / About / API

The State GeoNode is a digital service provided by the U.S. Department of State to provide open geographic data produced by or compiled by the U.S. Government to the public, U.S. Government decision-makers, and partners on complex emergencies, natural disasters, and diplomatic activities world-wide.

Developers can interact with the State GeoNode through 3 APIs: CSW, OGC Services, and the internal GeoNode API.

State GeoNode uses PyCSW to produce standardized CSW metadata services.

The CSW endpoint is: /catalogue/csw. To retrieve metadata records run something like this: /catalogue/csw?request=GetRecords&service=CSW&version=2.0.2&resultType=results&maxRecords=1000&elementsetname=full&typenames=csw:Record&outputschema=http://www.isotc211.org/2005/gmd. This endpoint can be used by other systems, such as Data.Gov and HDX, to automatically harvest records from the State GeoNode.

All layers in State GeoNode are accessible via OGC Services. The WMS Capabilities document is accessible with: /geoserver/wms?SERVICE=WMS&REQUEST=GetCapabilities. This endpoint can be used by geospatial analysts to pull layers directly into their desktop environment (/docs/integration.html). OGC services can also be used to pull layers directly from the state GeoNode into web maps to make mashups.
